# online-resumer

>## 初始化项目
使用vue-cli工具，vue-cli用到了webpack</br>
```
> mkdir online-resumer</br>
> cd online-resumer</br>
> npm init #可以生成package.json

```

全局安装vue-cli,并初始化一个vue项目</br>
```
> npm install -g vue-cli
> npm init webpack .
```
